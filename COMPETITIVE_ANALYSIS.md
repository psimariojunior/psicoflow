## Recommendations: Top 10 Features to Implement

### Tier 1 — Game Changers (implement first)

1. **AI Note Taker / Scribe**
   - Record therapy sessions (with patient consent)
   - Transcribe and generate SOAP/DAP notes using AI
   - Allow dictation mode (post-session voice notes → structured note)
   - Allow audio/text file upload → note generation
   - This is the #1 feature that SimplePractice, Jane, and Doctoralia all have
   - **Estimated effort**: High — requires audio recording, transcription API (Whisper), LLM integration (Hermes/OpenRouter), note template system

2. **AI Pre-Appointment Summaries**
   - Before each session, auto-generate a brief combining: last progress note, treatment plan, anamnesis highlights, recent questionnaire scores, recent emotion diary entries
   - Display on the session page
   - **Estimated effort**: Medium — uses existing data + LLM

3. **Digital Signature with Juridical Validity**
   - Allow psychologists to digitally sign prontuários, receitas, declarations
   - Allow patients to sign consent forms, treatment terms
   - Valid under Brazilian law (ICP-Brasil or PAdES)
   - **Estimated effort**: Medium — integrate with a signing API or implement crypto signing

4. **Document Templates with Auto-fill**
   - Create reusable document templates with placeholders like {patientName}, {date}, {crp}, {sessionCount}
   - Auto-generate documents (declarations, reports, referrals) from templates
   - **Estimated effort**: Medium — template engine + variable substitution

5. **Ratings & Reviews**
   - After completed sessions, email patients asking for a review
   - Display reviews on the public landing page (with patient consent/anonymized)
   - Use for SEO (structured data / rich snippets)
   - **Estimated effort**: Low-Medium — new model + public display + email

### Tier 2 — Revenue & Retention

6. **Packages & Memberships**
   - Sell packages (e.g., "10 sessions for R$1,200" — discount incentive)
   - Sell monthly memberships (e.g., "R$400/month includes 4 sessions")
   - Track package usage (sessions remaining)
   - Stripe integration for recurring billing
   - **Estimated effort**: Medium — new models + Stripe checkout modification

7. **Return Visit Reminders**
   - Automated check: patients who haven't booked in X weeks
   - Send personalized "We miss you" email/WhatsApp with booking link
   - Configurable interval (e.g., 30/60/90 days)
   - **Estimated effort**: Low — cron job + notification template

8. **Outcome Measure Tracking with Graphs**
   - Visualize questionnaire scores (PHQ-9, GAD-7) over time per patient
   - Line charts showing depression/anxiety severity over months
   - Show in patient profile and in session view
   - **Estimated effort**: Medium — charting library + data aggregation

9. **Patient Mobile App (PWA or React Native)**
   - PsicoFlow is web-only; a dedicated mobile app would improve engagement
   - Could start as a PWA (installable web app) for lower effort
   - Features: book sessions, view appointments, emotion diary, questionnaires, secure messaging, video calls
   - **Estimated effort**: Medium (PWA) / High (React Native)

10. **Corporate / B2B Portal**
    - Companies sign up and pay for their employees' therapy
    - Employees use PsicoFlow with a corporate code
    - Company dashboard: aggregate utilization, anonymized outcomes, ROI metrics
    - NR-1 compliance reporting
    - **Estimated effort**: High — new user type (Company), billing model, aggregate reporting

### Tier 3 — Nice to Have

- Group therapy sessions (LiveKit supports multi-participant)
- Async messaging therapy (between-session messaging)
- CID-10 lookup
- iCal calendar subscriptions
- Automated waitlist management (auto-offer cancelled slots)
- Email marketing campaigns
- Reserve with Google integration
- Gift cards
- ISO 27001 certification pursuit
- Multi-clinic support
- Professional commission splits
